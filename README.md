# Portfolio - RIN Nairith

A modern, responsive portfolio website showcasing my skills and projects as a Full Stack Developer & Data Scientist.

## 🚀 Features

- **Modern Design**: Glassmorphism effects, gradient animations, and neon accents
- **Responsive Layout**: Fully optimized for desktop, tablet, and mobile devices
- **Interactive UI**: Smooth scrolling, hover effects, and custom cursor
- **Dark Theme**: Elegant dark theme with enhanced readability

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Design**: CSS Grid, Flexbox, CSS Custom Properties
- **Icons**: Font Awesome 6
- **Fonts**: Inter (Google Fonts)

## 📂 Structure

```
├── index.html      # Main HTML file
├── styles.css      # Styling and animations
├── script.js       # Interactive functionality
└── README.md       # This file
```

## 🎯 Sections

- **Home**: Hero section with introduction and stats
- **About**: Professional background and highlights
- **Skills**: Technical skills with progress indicators
- **Projects**: Featured work and portfolio pieces
- **Blog**: Latest articles and insights
- **Contact**: Get in touch form and information
